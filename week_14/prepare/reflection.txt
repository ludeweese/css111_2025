1. Describe an experience you had this semester where you saw the power of
functions.
This semester, I learned that functions are a powerful tool for organizing and simplifying code.
In my final project, the President Oaks Conference Talks Manager, I organized the program into
multiple functions for tasks like loading talks, filtering by year or keyword, sorting, displaying,
and saving results. Using functions made the program reusable, easier to read, and simpler to debug,
showing how breaking a program into smaller tasks can manage complexity. Overall, this experience 
reflected the course lesson that each function should perform a single task, which helps programs 
stay clear and maintainable.

Writing tests also reinforced the course lessons on good coding practices, analytical thinking, 
and problem-solving, skills that will transfer to future programming courses and projects.
